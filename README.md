# Minimalist Rainmeter Skin
A custom-built Rainmeter suite focused on clustered data visualization and automation. This project was developed as part of my learning journey into scripting in **2023**. Feel free to use it as you like, though it's pretty much outdated and won't do anything cool.

## Features
- It is not minimalistic. Author is too lazy to come up with an actual name.
- Real-time displays for CPU, RAM, SWAP usage, and network adapter status.
- Customization is possible but tedious (not really).
- A lot of logical errors to find!

## Requirements
Ensure you have the latest version of Rainmeter installed.

## Installation
- Download the latest *.rmskin* file from the Releases page.
- Double-click the downloaded file to open the Rainmeter Skin Installer.
- Click Install.
- Once installed, open the Rainmeter manager to load the desired skins from this suite.
